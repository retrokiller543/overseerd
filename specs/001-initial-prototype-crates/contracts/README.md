# Contracts: Initial Prototype Crates

This directory documents the public contracts the first prototype must expose or preserve. These are source-level/library and demonstration contracts, not network contracts.

## Contract Files

- `core-registration.md`: Public behavior expected from the core metadata and explicit registration surface.
- `inspection-output.md`: Information that must be visible from the prototype inspection path.
- `crate-boundaries.md`: Responsibility boundaries for each planned crate.
